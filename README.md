# Virtusa Mini Projects

This repository contains mini projects developed as part of the Virtusa training program. These projects focus on different technologies and demonstrate practical implementation of concepts.

Currently, the repository includes Python, SQL, and Java projects.

## Projects Included

### 1. Python Project

Resume Analyzer and Job Matcher

This is a web-based application built using Flask that compares a resume with a job description. It calculates a match score, identifies missing skills, and provides suggestions to improve the resume.

Key features:

* Resume and job description comparison
* Match score generation
* Skill extraction and missing skill detection
* Suggestions for improvement
* History tracking using SQLite

Technologies used: Python, Flask, SQLite, NLTK, Scikit-learn, HTML, CSS


### 2. SQL Project

Hospital Management and Patient Analytics System

This project is a database system that stores hospital-related data and performs analysis on it. It manages patients, doctors, appointments, and treatments.

Key features:

* Patient and doctor data management
* Appointment tracking
* Treatment and cost recording
* Analytical queries for insights

Technologies used: SQL, Relational Database Concepts



### 3. Java Project

Banking System Simulation

This project is a console-based banking system developed using Core Java. It simulates real-world banking operations and demonstrates object-oriented programming concepts.

Key features:

* Create Savings and Current accounts
* User authentication using username and password
* Deposit, withdraw, and transfer money
* Balance inquiry
* Transaction history tracking

Technologies used: Java, Object-Oriented Programming, Java Collections Framework

## How to Use

Clone the repository and navigate to each project folder.

### For the Python project:

* Create a virtual environment
* Install dependencies
* Run the Flask application

### For the SQL project:

* Execute the SQL scripts in a database environment

### For the Java project:

* Compile the program using javac
* Run the Main class



## Conclusion

These projects demonstrate practical skills in backend development, database design, object-oriented programming, and data analysis. The repository showcases the ability to work across multiple technologies and build real-world applications.
